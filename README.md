# 🏥 MediGuide AI

**Offline RL-Powered Emergency Medical Assistant**

> Meta + Hugging Face Hackathon 2026 · Healthcare Track

## Features
- 🩺 Medical diagnosis based on symptoms
- 🚨 Emergency condition detection
- 🇮🇳 Hindi + English support
- 📱 Works offline (rule-based engine)

## Usage
Enter symptoms to get AI-powered diagnosis and emergency guidance.

## Environment Variables
- `HF_TOKEN` - HuggingFace token for inference API (optional)
- `MODEL_NAME` - Model to use (default: google/gemma-2b-it)

## Disclaimer
This is AI guidance only. Always consult a qualified doctor for medical decisions.
